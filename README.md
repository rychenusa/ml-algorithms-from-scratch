# ML Algorithms and Interpretability

Collection of machine learning algorithm implementations and model interpretability experiments.

The notebooks focus on building core algorithms from scratch and analyzing how models learn, rather than relying solely on high-level libraries.

Topics include:

• stochastic gradient descent and least mean squares (LMS)  
• neural networks trained from scratch  
• Grad-CAM visualization for convolutional neural networks  
• model behavior on toy datasets (e.g., XOR)

These experiments were originally developed while studying machine learning and optimization, with an emphasis on understanding algorithm mechanics.

---

## Notebooks

**lms_stochastic_gradient.ipynb**  
Implementation of the Least Mean Squares algorithm using stochastic gradient descent.

**two_layer_nn_xor_from_scratch.ipynb**  
Two-layer neural network trained from scratch to learn the XOR function.

**gradcam_expression_cnn.ipynb**  
Grad-CAM visualization applied to a CNN trained for facial expression recognition.

**gradcam-visualization.ipynb**  
Exploration of Grad-CAM heatmaps to interpret convolutional neural network predictions.

---

## Focus

The goal of this repository is to explore:

• optimization dynamics in machine learning  
• neural network training behavior  
• interpretability techniques for deep learning models
